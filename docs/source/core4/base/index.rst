####
base
####

Package :any:`core4.base <core4.base.main>` with class
:any:`core4.base.CoreBase <core4.base.main.CoreBase>`
represent the basis to all core4os implementation. All classes derived from
:class:`.CoreBase` ship with access to configuration, data (see also
:doc:`data access protocols <connector>`), and logging (see
also :doc:`logging mechanics<../logger/index>`) among other features.
